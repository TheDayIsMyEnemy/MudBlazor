# MudBlazor Template
Blazor Template using [MudBlazor](https://github.com/MudBlazor/MudBlazor/)

## Blazor Server

### Content
- [Main branch](https://github.com/TheDayIsMyEnemy/MudBlazorTemplate/tree/main) includes only the Dark Theme
