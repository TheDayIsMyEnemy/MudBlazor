# MudBlazor Template
Blazor Template using [MudBlazor](https://github.com/MudBlazor/MudBlazor/)

## Blazor Server

### Content
- [main branch](https://github.com/TheDayIsMyEnemy/MudBlazorTemplate/tree/main) has only dark theme
